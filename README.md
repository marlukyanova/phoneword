## Installation

1. Clone this repo and enter!

   ```bash
   git clone https://github.com/marlukyanova/phoneword.git
   cd phoneword
   ```

2. Install dependencies for server.

   ```bash
   cd server
   npm install
   nodemon
   ```

3. Install dependencies for client.
   
   ```bash
   cd phoneword
   npm install
   npm start
   ```
